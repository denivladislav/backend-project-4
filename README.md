## How To Use
```bash
# Install dependencies.
$ npm install

# Link the package.
$ npm link

# Get info.
$ page-loader -h 

# Start tests / tests and show coverage / tests with debug mode
$ make test
$ make test-coverage
$ make test-debug
```
